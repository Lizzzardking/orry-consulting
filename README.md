## 🎯 Project Overview
This is a real-world platform built for **Orryana** (Finance Influencer, 30k+ followers). The mission is to transition her digital presence from monolithic **WordPress environments** to a high-performance, **Headless React architecture**.

## 🏗 The Evolution: Beyond the CMS
With over **13 years of experience as a Senior Designer and Web Developer**, I am not new to code. My background includes years of building custom web solutions within the **WordPress ecosystem**. 

However, this project marks my strategic shift from "CMS-driven development" to **Modern UI Engineering**:
- **Technical Refactoring:** Moving away from PHP-based monolithic structures towards a decoupled, Headless approach.
- **Engineering Mindset:** Replacing "plug-in thinking" with custom, logic-heavy React components and scalable architecture.
- **Design Tokens:** Bridging my deep branding expertise with systematic code (Tailwind v4).

## 🛠 Technical Stack
- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS v4.0 (Custom design system integration)
- **Previous Stack:** 13+ years of HTML/CSS, PHP, and WordPress customization.
- **Current Focus:** Masterful State Management, API integration, and Component Logic.

## ✨ Strategic Features
- [x] **Brand-to-Code Pipeline:** Full translation of Figma design systems into a Tailwind v4 theme.
- [x] **Wealth Builder (MVP):** An interactive investment calculator focusing on **Derived State**—leveraging my logic experience in a modern React environment.
- [ ] **Content Engine:** Dynamic data fetching from a Headless CMS (Sanity.io) to replace traditional WordPress data structures.

## 🚀 Why this project?

This repo documents my journey of taking **13 years of industry knowledge** and re-tooling it for the **Modern Web Stack**. I focus on professional Git workflows, clean React patterns, and performance-first engineering.
